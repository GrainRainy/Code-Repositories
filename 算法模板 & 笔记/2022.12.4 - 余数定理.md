# 三大余数定理

## 余数加法定理

`(a + b) % c = (a % c + b % c) % c`

**证明：**

若 `a = xc + p`, `b = yc + q`
则 `a % c = p`, `b % c = q`

则有:
`(a + b) % c = [(x + y)c + p + q] % c = (p + q) % c`
`(a % c + b % c) % c = (p + q) % c`

## 余数乘法定理

`(a * b) % c = (a % c * b % c) % c`

## 同余定理

当且仅当两个数 $a$ 和 $b$ 满足：$(a - b)\ \%\ m == 0$ 我们称 $a$ 与 $b$ 同余。

形象化地，有 $a\ \%\ m\ ==\ b\ \%\ m$

记作：$a \ \equiv\ b$
