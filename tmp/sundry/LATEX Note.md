# 常用 $\LaTeX$ 代码

## 运算

- 叉乘：$\times$
- 点乘：$\cdot$
- 除法：$\div$
- 整除：$\mid$
- 模运算：$\bmod$
- 同余：$\equiv$
- 分式：$\frac{1}{2}$
- 整除：$\mid$

## 二元关系符 $Binary\ Relations$

- 不等于： $\ne$
- 小于： $<$
- 小于等于：$\le$
- 斜小于等于：$\leqslant$
- 大于：$>$
- 大于等于：$\ge$
- 斜大于等于：$\geqslant$

## 集合运算

- 属于：$\in$
- 反属于：$\ni$
- 包含于：$\subset$

## 大型运算

- 求和：$\sum_{}^{}$

## 括号

- 上取整：$\lceil \ \rceil$
- 下取整：$\lfloor \ \rfloor$

## 特殊符号

- ~，波浪线：$\sim$
- $\sout{删除线}$：$\sout{}$
- $\bcancel{斜删除线}$：$\bcancel{}$
- $\cancel{反删除线}$：$\cancel{}$
- $\underline{下划线}$：$\underline{}$
